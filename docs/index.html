Student Survival Manager

Academic Planner + Expense Tracker

This project is a JavaFX desktop app for managing assignments, events, and expenses.

View Source Build Artifacts

How to Use

GitHub Pages hosts this project website. The app itself runs as a desktop Java application and cannot run directly in the browser.

  1. Download the built JAR from GitHub Actions artifacts.
  2. Install Java 17 or newer.
  3. Run the app locally.
mvn clean javafx:run

Assignments

Track due dates, priorities, and completion status.

Expenses

Record spending and monitor your running total.

Events

Plan events with date, location, and completion status.
